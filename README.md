## Installation

Clone and install the necessary packages

```bash
git clone https://github.com/sojida/hux-assessment-backend
cd hux-assessment-backend
npm install
```

## Usage

To start the API

- In Dev mode
```bash
npm run start:dev
```

- In Production
```bash
npm run start:dev
```

## Features

Features for this project includes:
- Sign up
- Login
- Create contact
- Update contact
- Delete contact
- Get contact(s)
